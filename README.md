![](assets/slogan.png)

一站式前端工程解决方案。
---

功能包括：
* 脚手架；
* 基于webpack的编译系统；
  * ES6/7；
  * css预编译&postcss；
  * 图片压缩&base64嵌入；
  * 模块化:ES6 Modules&CommonJS&AMD；
* 多页面开发
* 一键部署
  * 部署测试环境；
  * 部署生产环境。
* 本地服务器
  * mock服务；
  * 动态编译；
  * 浏览器自动刷新。
* boi扩展
  * boi插件系统；
  * 可定制脚手架；
  * 自定义代码规范。
